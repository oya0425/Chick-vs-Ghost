# Chick vs Ghost(ひよこ vs お化け)
DirectX11 / C++ で制作した、スピード感あふれる3Dアクションゲームです。
<img width="1602" height="901" alt="スクリーンショット 2026-06-06 185223" src="https://github.com/user-attachments/assets/e2c4249e-507c-4f9b-b4f5-06a643bfbe0d" />

プレイヤーの戦術をリアルタイムに「学習・進化」させる敵の群れに立ち向かう、戦略的ローグライク・アクションゲームです。
たくさん敵を倒して自身を強化し、プレイヤーの行動によって強化されたボスを５回撃破しよう！

---

## 🎮 ゲームシステム（コアループ）
1. **敵を討伐・レベルアップ**：スキルを駆使して大量の敵を倒し、プレイヤーのステータスを強化！
2. **敵のリアルタイム学習**：プレイヤーの倒し方に応じて、敵の「リーダー」が戦術を学習し進化。
3. **ボスの超強化（最終WAVE（WAVE5））**：それまでに倒した全リーダーの学習データを合計・吸収したボスが登場！
4. **クリア条件**：徐々に拡大していくステージで、進化するボスを合計 **5回撃破** すればゲームクリア。

## 👻 敵（群れ行動 ＆ 状態変化）
敵はお化けの「群れ（グループ）」として行動します。群れには必ず **1体のリーダー** が存在し、周囲の雑魚敵はリーダーに追従します。
リーダーを先に撃破すると、残された群れは統率を失い、以下のどちらかの特殊状態に変化します。

* 🌀 **パニック状態**
  * 恐怖で逃げ惑い、一定時間後に新しいリーダーを探し始めます。この状態の敵を倒すと**獲得経験値が大幅にアップ**します。
* 🔥 **特攻状態**
  * 狂暴化し、プレイヤー目掛けて一斉に突撃してきます。接触するとダメージを受けますが、**「範囲攻撃スキル」で迎え撃てばノーダメージ**で一掃できます。

### 【経験値システム】
* **リーダーお化け**：討伐時、通常よりも多くの経験値を獲得。
* **通常の雑魚お化け**：「パニック状態」の時に倒すことで、通常より多くの経験値を獲得。

## 🧠 敵の学習 ＆ ボスWAVEへのデータ引き継ぎ

### 1. リーダーの学習（進化）

リーダーが倒される際、その「倒され方」に応じて、次にスポーンするリーダーが恒久的に成長します。

| プレイヤーの行動 | リーダーの学習効果 |
| :--- | :--- |
| **近接攻撃で倒す** | 群れが「特攻状態」になる確率アップ　|
| **範囲攻撃で倒す** | 基礎速度アップ |
| **引き寄せ攻撃で倒す** | 引き寄せを確率で無効化する（受けるたびに無効化確率が累積上昇） |

### 2. ボスWAVE（最終決戦）
最後のWAVEには、それまでの戦術の「ツケ」が全て回ってきたボスが単体で登場します。
ボスは、**これまでにプレイヤーが倒してきた全てのリーダーの学習データを凝縮・合成**して自身のステータスにします。

* 🏃‍♂️ **近接攻撃耐性** ➔ 全リーダーの合計値を、自身が倒されるときにプレイヤーへダメージを与える。
* 📏 **範囲攻撃耐性** ➔ 全リーダーの合計値を、自身の移動速度にそのまま加算（高速移動）。
* 🛡️ **引き寄せ攻撃耐性** ➔ 過去のリーダー達の中で**「最も高い無効化確率」**をそのまま継承。

## 🛠️ その他のシステム機能
* 📋 **リアルタイム・デバッグUI（TABキー）**
  * ゲーム中に `Tab` キーを押すことで、現在戦っている群れやボスの学習数値（耐性）を視覚的な棒グラフでいつでも確認可能。？マークにマウスを合わせると滑らかなイージングアニメーションで詳細テキストがポップアップします。
* 💬 **群れメッセージ確認（1 / 2 / 3 / 4 キー）**
  * 各キーを押すことで、対応する群れの状態メッセージを確認できます。
* 🗺️ **動的ステージ拡大**
  * WAVEが進むごとに、戦いの舞台となるステージがどんどん広大に変化していきます。
